# Historia Danet — Build APK via GitHub Actions

## Cara Build APK (Tanpa Install Apapun!)

### Step 1 — Buat Repository GitHub
1. Buka https://github.com → Login → klik "New repository"
2. Nama: `historia-danet` → pilih Public → Create

### Step 2 — Upload File
Di repo kosong → klik "uploading an existing file"
Upload SEMUA file dari folder ini termasuk folder .github/

### Step 3 — Tunggu Build
Klik tab "Actions" → tunggu ±10-15 menit → hijau = berhasil

### Step 4 — Download APK
Actions → klik workflow selesai → Artifacts → HistoriaDanet-APK → download

### Step 5 — Install ke HP
Copy APK ke HP → tap untuk install → selesai!
